# Данные для обучения модели

## UCI Cardiotocography Dataset

Модель обучается на открытом датасете UCI CTG, содержащем 2126 КТГ-записей с
21 FIGO-признаком и экспертной оценкой 3 неонатологов.

### Скачивание

1. Перейди по ссылке: https://archive.ics.uci.edu/dataset/193/cardiotocography
2. Нажми **Download** → скачается архив `cardiotocography.zip`
3. Распакуй и найди файл `CTG.xls` (лист `Raw Data`)
4. Положи файл в эту директорию: `backend/data/CTG.xls`

### Запуск обучения

```bash
cd backend
python ml/train.py
```

После обучения в `backend/ml/models/` появятся:
- `catboost_v1.cbm`  — обученная модель
- `metrics.json`     — метрики на тестовой выборке

### Структура датасета

| Признак | Описание |
|---------|----------|
| LB      | Базальная ЧСС плода (уд/мин) |
| AC      | Число акцелераций |
| FM      | Движения плода |
| UC      | Маточные сокращения |
| ASTV    | Краткосрочная вариабельность (%) |
| MSTV    | Средняя краткосрочная вариабельность (мс) |
| ALTV    | Долгосрочная вариабельность (%) |
| MLTV    | Средняя долгосрочная вариабельность (мс) |
| DL      | Поздние децелерации |
| DS      | Тяжёлые децелерации |
| DP      | Пролонгированные децелерации |
| DR      | Повторяющиеся децелерации |
| Width   | Ширина гистограммы |
| Min     | Минимум гистограммы |
| Max     | Максимум гистограммы |
| Nmax    | Пиков гистограммы |
| Nzeros  | Нулей гистограммы |
| Mode    | Мода гистограммы |
| Mean    | Среднее гистограммы |
| Median  | Медиана гистограммы |
| Variance| Дисперсия гистограммы |
| **NSP** | **Целевая переменная: 1=Normal, 2=Suspect, 3=Pathological** |

### Распределение классов

| Класс | Кол-во | % |
|-------|--------|---|
| Normal (1)       | 1655 | 77.8% |
| Suspect (2)      | 295  | 13.9% |
| Pathological (3) | 176  | 8.3%  |

> Дисбаланс компенсируется через `class_weights` в CatBoost.
