# Примеры данных для загрузки

Эти файлы можно загружать через страницу **«Загрузить данные»** на сайте
или через API (`POST /upload/*`).

---

## 1. ctg_features_example.csv — КТГ-признаки (прямой предикт)

**Формат:** 21 FIGO-признак в одной строке + опциональный `NSP` (истинный класс).

**Эндпоинт:** `POST /upload/ctg-features`

| Колонка | Диапазон | Описание |
|---------|---------|----------|
| LB      | 50–200  | Базальный ритм ЧСС плода (уд/мин) |
| AC      | 0–20    | Число акцелераций |
| FM      | 0–20    | Движения плода |
| UC      | 0–10    | Маточные сокращения |
| ASTV    | 0–100   | Краткосрочная вариабельность (%) |
| MSTV    | 0–10    | Средняя STV (мс) |
| ALTV    | 0–100   | Долгосрочная вариабельность (%) |
| MLTV    | 0–50    | Средняя LTV (мс) |
| DL      | 0–20    | Поздние децелерации |
| DS      | 0–10    | Тяжёлые децелерации |
| DP      | 0–10    | Пролонгированные децелерации |
| DR      | 0–5     | Повторяющиеся децелерации |
| Width   | 0–200   | Ширина гистограммы |
| Min     | 50–150  | Минимум FHR |
| Max     | 100–250 | Максимум FHR |
| Nmax    | 0–20    | Пиков гистограммы |
| Nzeros  | 0–100   | Нулей гистограммы |
| Mode    | 50–200  | Мода гистограммы |
| Mean    | 50–200  | Среднее гистограммы |
| Median  | 50–200  | Медиана гистограммы |
| Variance| 0–1000  | Дисперсия гистограммы |

Пример строки (Normal):
```
133,0,0,0,27,0.9,27,6.8,0,0,0,0,64,62,126,2,18,120,137,121,73,1
```

## 2. ctg_signals_example.csv — Сырые КТГ-сигналы (с извлечением признаков)

**Формат:** Два числовых столбца: `fhr` (ЧСС плода, уд/мин) и `uc` (маточная активность).  
Минимальная длина: **1200 строк** (5 мин при частоте 4 Гц).

**Эндпоинт:** `POST /upload/ctg-signals?fs=4`

```csv
fhr,uc
138.2,0.00
139.1,0.12
137.8,0.45
...
```

Система автоматически:
1. Очищает сигнал (IQR-выбросы, интерполяция пропусков)
2. Извлекает 21 FIGO-признак
3. Возвращает предсказание класса

## 3. ecg_maternal_example.csv — Показатели матери (ЭКГ/витальные)

**Формат:** 6 числовых признаков материнского здоровья.

**Эндпоинт:** `POST /upload/ecg-maternal`

| Колонка     | Диапазон | Описание |
|-------------|---------|----------|
| Age         | 10–65   | Возраст (лет) |
| SystolicBP  | 60–250  | Систолическое АД (мм рт.ст.) |
| DiastolicBP | 40–180  | Диастолическое АД |
| BS          | 1–30    | Сахар крови (ммоль/л) |
| BodyTemp    | 35–42   | Температура тела (°C) |
| HeartRate   | 40–200  | ЧСС матери (уд/мин) |

Опциональный столбец `RiskLevel` (low risk / mid risk / high risk) — для верификации.

Пример строки (high risk):
```
38,150,100,13.0,37.8,88,high risk
```

## 4. WFDB-запись (ZIP)

**Формат:** ZIP-архив, содержащий `.dat` + `.hea` файлы в формате PhysioNet.

**Эндпоинт:** `POST /upload/wfdb`

Скачать тестовые записи: https://physionet.org/content/ctu-uhb-ctgdb/

```
archive.zip
├── 1001.dat
└── 1001.hea
```

---

## Скачать примеры через API

```bash
# CTG признаки
curl http://localhost:8000/upload/examples/ctg_features -o ctg_features_example.csv

# CTG сигналы
curl http://localhost:8000/upload/examples/ctg_signals -o ctg_signals_example.csv

# Материнские показатели
curl http://localhost:8000/upload/examples/ecg_maternal -o ecg_maternal_example.csv
```
